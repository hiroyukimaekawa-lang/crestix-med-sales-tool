# crestix-med-sales-tool

Crestix Inc. 第一営業部 — 医療クリニック向け営業業務ツールポータル

## 概要

第一営業部で使用する社内ツールをまとめたWebポータルです。  
CSVをアップロードするだけでブラウザ上で処理が完結します（サーバーへのデータ送信なし）。

## ツール一覧

| ツール | パス | 状態 |
|---|---|---|
| マイナビ重複判定 | `/dedup/` | 利用可能 |

## ディレクトリ構成

```
crestix-med-sales-tool/
├── index.html        ← ツールポータル（トップ）
├── dedup/
│   └── index.html    ← マイナビ重複判定ツール
└── README.md
```

## 新しいツールの追加方法

1. 新しいフォルダを作成（例：`banner/`）
2. `index.html` を追加
3. トップの `index.html` にツールカードを追加
4. GitHubにプッシュ → Vercelに自動デプロイ

## デプロイ

Vercel（GitHub連携）で自動デプロイ。  
`main` ブランチへのプッシュで本番環境に反映されます。

## 注意事項

- 社内限定ツールです
- データはすべてブラウザ内で処理され、外部に送信されません
