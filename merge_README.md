# 3媒体統合ツール

## 概要
DF・MD・マイナビ・DF_MDの媒体CSVをアップロードし、重複統合・媒体別分類を行うWebツール。

## 使い方
1. DF課金リスト・MD課金リスト・マイナビのCSVをアップロード（1ファイルから実行可能）
2. 「統合を実行」ボタンを押す
3. 7つのタブで結果を確認
4. 必要なタブのCSVをダウンロード

## 入力ファイル

| ファイル | 主要ヘッダー |
|---|---|
| DF課金リスト.csv | 医院名・電話番号・住所・医師名・営業曜日・クリニックURL |
| MD課金リスト.csv | 医院名・電話番号・住所・記事タイトル・営業曜日・ホームページ |
| マイナビ.csv | UUID・名前・Tel1・都道府県・住所１・記事名 他 |

- 文字コード：Shift-JIS / UTF-8（BOM有無）自動判定

## 出力ファイル（7種類）

| ファイル名 | 内容 |
|---|---|
| DF_MD_マイナビ.csv | 3媒体すべてに存在 |
| DF_MD.csv | DFとMDに存在 |
| DF_マイナビ.csv | DFとマイナビに存在 |
| MD_マイナビ.csv | MDとマイナビに存在 |
| DFのみ.csv | DFのみ |
| MDのみ.csv | MDのみ |
| マイナビのみ.csv | マイナビのみ |

## 重複判定ロジック
1. 電話番号一致（全角→半角・数字のみ・先頭0除去）
2. 電話番号が空の場合のみ住所一致

## マージ優先順位
マイナビ > DF > MD
